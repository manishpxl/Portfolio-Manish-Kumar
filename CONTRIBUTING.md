# Contributing to Manish Portfolio

Thank you for your interest in contributing to this portfolio project.

This repository contains a simple and clean developer portfolio built using HTML, CSS and JavaScript. Contributions that improve performance, accessibility, UI or code quality are welcome.

---

## 🚀 How to Contribute

### 🐞 Reporting Issues

If you find any bug or issue, please create an issue with:

* Clear description of the problem
* Steps to reproduce
* Expected vs actual behavior
* Screenshot (if helpful)
* Browser or device details

---

### ✨ Suggesting Improvements

You can suggest:

* UI/UX enhancements
* Performance optimizations
* Accessibility improvements
* New section ideas

Please describe the benefit and possible approach.

---

### 🔧 Submitting a Pull Request

1. Fork the repository
2. Create a new branch
   `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test your updates properly
5. Commit with a clear message
   `git commit -m "Improve portfolio UI"`
6. Push your branch
   `git push origin feature/your-feature-name`
7. Open a Pull Request

---

## 📌 Code Guidelines

### HTML

* Use semantic elements
* Maintain proper indentation
* Add comments for complex sections

### CSS

* Follow existing structure
* Use variables for colors
* Keep responsive design intact

### JavaScript

* Keep logic simple and readable
* Avoid unnecessary global variables

---

## ✅ Before Creating PR

* Portfolio runs without errors
* Responsive layout is not broken
* No console warnings
* Performance is not affected
